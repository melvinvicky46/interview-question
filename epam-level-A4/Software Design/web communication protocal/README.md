**Skill Description**
Web communication protocols are technologies used to transfer information across the internet. For example, a web browser uses these protocols to request information from a web server, which is then displayed on the browser screen in the form of text and images.  

**What you should know A4**
Advanced knowledge of:

HTTP vs HTTPS vs HTTP/2
WS vs Polling
RESTful vs RPC (JSON RPC) vs GraphQL 

**What you should know A3**
Intermediate knowledge of:

HTTP vs HTTPS vs HTTP/2
WS vs Polling
RESTful API
Nice to have:

RPC (JSON RPC)
GraphQL
